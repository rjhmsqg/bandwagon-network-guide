## 搬瓦工荷兰机房和加拿大机房是什么

先把基本概念铺一下，方便后面对比。

**搬瓦工荷兰 EUNL_9 机房**：位于荷兰阿姆斯特丹，代号 EUNL_9，接入的是联通 AS9929 + AS10099 精品政企线路（俗称联通 9929），这是中国联通对外的顶级商用线路，比普通联通 169 骨干网高一个档次。三网方向都走这条线路，但体验差异比较大——联通用户是主要受益方，电信和移动绕路稍多。

搬瓦工还有另外两个荷兰机房：EUNL_3（普通线路，适合面向欧洲用户）和 EUNL_1（新机房，三网回程接入 CN2 GIA，仅限 CN2 GIA-E 套餐新购时选择，买完不建议切出去）。三个机房线路完全不同，别搞混。

**搬瓦工加拿大温哥华 CABC_6 机房**：代号 CABC_6，接入电信 CN2 GIA-E + 移动 CMIN2 + 联通 CUP，是标准的三网高端线路覆盖。和洛杉矶 DC6/DC9 同属 CN2 GIA 系列，但物理位置在温哥华，对加拿大或西海岸有业务的用户更友好，国内延迟和洛杉矶大体相当。

EUNL_9 是给联通用户准备的，特别是那种跑国际业务、欧洲有客户、或者就是偏好联通精品线路的需求。电信用户冲 EUNL_9 不太划算，绕得有点远。
